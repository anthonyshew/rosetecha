# Rate limit

Rate limits are when an application stops someone from requesting the same information many times in quick succession.

This technique is important because it stops a hacker from repeatedly hitting your app's server so often that they slow down other traffic. See <KeyTermLink termUrl="ddos">DDoS</KeyTermLink> for mor information

For more information, see [Rate Limiting](https://www.cloudflare.com/learning/bots/what-is-rate-limiting).
