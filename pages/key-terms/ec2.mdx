# EC2

ECS is an AWS service that provides a computer that you can run your code on.

This computer lives out in Amazon's data centers and can be configured to perform all sorts of tasks, including being a home for website.

For more information, see [Amazon Elastic Compute Cloud](https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/concepts.html).
