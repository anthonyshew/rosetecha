# Amazon Global Accelerator

AGA is a networking solution offered by AWS that speeds up the delivery of websites when the global internet network is working sluggishly. It works by making sure that the paths that a request for your website remain free and clear of any noise.

For more information, see [Amazon Global Accelerator Developer Guide](https://aws.amazon.com/global-accelerator/).
